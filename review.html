CS/IT 490 WD

Web Development

Course description: In this course, the student will learn the basics of open source internet programming, to include: HTML, CSS, PHP, MySQL and JavaScript/AJAX. Students will create dynamic websites utilizing user registration/login, database access and style documents. There will be one midterm exam and one final exam along with a course project chosen by the students. Prerequisite: CS 101, equivalent or consent of the instructor. This course is cross-listed as CS/IT 490WD. Regular Academic.
